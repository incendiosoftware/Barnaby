# Security Policy

## Supported Versions

Security fixes are provided for the latest release on `main`.

## Reporting a Vulnerability

Do not open a public issue for security vulnerabilities.

Please report vulnerabilities privately using GitHub's security advisory flow:

- Go to the Security tab for this repository.
- Click "Report a vulnerability".
- Include reproduction steps, affected versions, and impact.

If you cannot use the advisory flow, contact the maintainer directly and include "SECURITY" in the subject.
